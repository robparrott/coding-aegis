# External dependency installation for packages

**Status**: open | **ID**: `coding-aegis-a0q`

Design and implement how coding-aegis packages can declare and install external dependencies — skills, agents, or configs authored outside this catalog. First use case: fancy-beads should pull in the beads issue-tracking skill from sttts/beads-skill (https://github.com/sttts/beads-skill). This differs from AD-12 external package references (which are catalog pointers to external repos). This is about a package in our catalog depending on artifacts from an upstream project.
